Site Criativo:Alura Start

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma turma do Alura fictícia de alunos e Alunas, além de explicar um pouco sobre o alurastart.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de alunos e alunas em uma seção flexível. Cada aluno é destacado em seu próprio contêiner, mostrando uma imagem genérica de profile, além de ícones de contato.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando saber mais de cada aluno.

Acessibilidade na Web

Este projeto é dedicado a demonstrar e implementar as melhores práticas de acessibilidade web, seguindo as diretrizes WCAG (Web Content Accessibility Guidelines).

O foco está em garantir que todos os elementos do site sejam acessíveis para pessoas com diferentes tipos de deficiência, incluindo recursos como:

  • Navegação por teclado
  • Contraste adequado de cores
  • Textos alternativos para imagens
  • Estrutura semântica HTML
  • Suporte a leitores de tela

O projeto utiliza HTML5 semântico, ARIA attributes quando necessário e CSS que mantém a legibilidade em diferentes condições de visualização.

Gerador de Senhas Seguras

Uma aplicação web para geração de senhas seguras desenvolvida com JavaScript, focada em segurança digital e usabilidade.

Recursos principais:

  • Personalização do comprimento da senha (8-64 caracteres)
  • Opções para incluir maiúsculas, minúsculas, números e símbolos
  • Indicador de força da senha em tempo real
  • Funcionalidade de copiar para a área de transferência
  • Geração baseada em algoritmos criptograficamente seguros

O projeto utiliza o método crypto.getRandomValues() para garantir a aleatoriedade das senhas geradas, implementando boas práticas de segurança desde o desenvolvimento.
